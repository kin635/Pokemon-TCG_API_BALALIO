How to Use Pokémon TCG Browser
Quick Start

Open the application:
-Simply open index.php in your web browser
-No server setup required!

Create an account:
-Click "Sign Up" on the login screen
-Enter your name, email, and password
-Click "Sign Up" to create your account

Start browsing cards:
-Use the search bar to find specific Pokémon
-Apply filters by type, rarity, or sort options
-Click any card to view detailed information

Main Features
Searching & Filtering
Search: Type any Pokémon name in the search bar
Type Filter: Filter by Grass, Fire, Water, etc.
Rarity Filter: Show Common, Uncommon, Rare cards
Sort Options: Sort by name, power, value, or HP

Collection Management
Add to Collection: Click any card → "Add to Collection"
View Collection: Check "Show My Collection" toggle
Remove Cards: Use "Remove from Collection" in card details

Profile & Stats
View Profile: Click "Profile" in navigation
Collection Stats: See total cards, value, and average power
Login History: Track your account activity

Navigation
Home: Browse and search cards
About: Learn about the application
Contact: Send messages (demo feature)
Profile: View your account and collection stats

Tips
Cards automatically save to your browser's storage
Your collection persists between sessions
Use pagination to browse through large result sets
Click the user avatar to quickly access your profile

Troubleshooting
Cards not loading? Check your internet connection
Login issues? Ensure password is 8+ characters with letters and numbers
Display problems? Refresh the page or clear browser cache
